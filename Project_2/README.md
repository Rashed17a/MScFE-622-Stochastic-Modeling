## Topic

**Time Series Analysis & Regime Detection** — Covers the analysis of financial time series for market regime shifts using adaptive modeling techniques applied to equity and fixed income data.

## Key Topics

- Financial time series data collection (FRED API)
- S&P 500 and Treasury yield analysis
- Market regime shift detection
- Hidden Markov Models (HMM) for regime identification
- Adaptive finance models for regime-dependent portfolio management
