## Topic

**Risk-Aware Portfolio Selection (Multi-Armed Bandit)** — Covers sequential portfolio optimization using a risk-aware Multi-Armed Bandit (MAB) framework that integrates Conditional Value-at-Risk (CVaR) to balance return and downside risk.

## Key Topics

- Multi-Armed Bandit (MAB) framework for portfolio selection
- UCB1 policy for exploration-exploitation tradeoff
- Conditional Value-at-Risk (CVaR) for risk quantification
- Convex optimization for risk-minimizing portfolios
- Cholesky decomposition and correlated asset simulation
- Comparison of UCB1, CVaR, and combined portfolio strategies
